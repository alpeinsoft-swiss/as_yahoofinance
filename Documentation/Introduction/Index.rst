.. _what-it-does:

What does it do?
----------------

The Extension provides a plugin to show financial statistics from a YahooFinance channel.

.. _features:

Features
--------

- Use of Extbase / fluid Written
- Working with Chart service of YahooFinance
- Use of amChart for graphic data presentation
- Data processing for different time intervals
- Scalable data
- Time indicator for each market value
- Customizable interface
- A set of standard views
